# Sharky

My first Unity project, built in 2022 when I was starting out with C# and Unity. A simple 2D arcade game — control a shark, eat fish and avoid rocks.

I've kept this repo as a starting-point reference. For more recent work, see my [Flappy Bird Demo](https://github.com/kadirdurakli/flappy-bird-demo).
